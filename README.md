### Introduction
与同学一起研究的一道有趣的题目：
> 有一个数列 $\{1, 2, ..., 2018\}$，每次操作可以从数列中取出两个数（不放回），将这两个数相加并把和的各个数字之和插入到原数列。问若干次操作后剩下的最后一个数的最大值和最小值。

#### 机房
- 题目讨论
- 估算答案
- 证明
- 完成了 `C++` 语言的求解代码

#### 家
- 将代码移植为 `Python` 库
- 基于  `PyWebIO` 完成了在线的用户界面

---

#### 优点
- 速度快（复杂度较优秀）
- “用户界面简单易用”

#### 缺点与不足
- 无法解决更大的数（解法弹性不足）
